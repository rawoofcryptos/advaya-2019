DESCRIPTION

      Two person relay of coding world. Coding in half and half without being in the same Room. Two halves by two different coders only with limited time to communicate while passing the baton.

RULES

  1. Team event (2 persons per team).
  2. Consists of two rounds.
  3. First round will consist of 30 question based on c language.
  4. Based on the results of first round the participants will be selected for second round.
  5. In the second round problem statement will be provided to the first person only.
  6. First person need to type the code for the first half.
  7. Second person need to type the code for next half and complete the code.
  8. There will be chance to communicate while passing baton.
  9. There will be a time limit.
  10. Winners will be announced based on the results of compilation and execution.
