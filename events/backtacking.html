DESCRIPTION

      It will be fun if you are coding with a piece of flowchart that your friend has been created.As he completes the flowchart you will complete the code...Lets Track back...From code to code...

RULES

  1. There will be a team of 2 people.
  2. One of the team member will get a program source code with which he/she has to design a flow chart in 10 minutes.
  3. Then the other teammates will code in computer with the flowchart of his/her teammates.
  4. The cycle will repeat for 2 or 3 times.
  5. The code with least errors will win the prize.
